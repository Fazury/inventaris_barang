# Sistem Inventaris Barang

Aplikasi CRUD berbasis web untuk manajemen inventaris barang.

## Fitur
- CRUD Barang
- CRUD Kategori Barang
- Pencarian & Filter
- Export ke Excel dan PDF
- Pagination
- Validasi Input
- Antarmuka Bootstrap

## Instalasi
1. Clone repositori:
2. Jalankan XAMPP
3. Import database `inventaris.sql` via phpMyAdmin
4. Akses via browser:

## Struktur Folder
- `config/` - Koneksi DB
- `barang/` - CRUD barang
- `kategori/` - CRUD kategori
